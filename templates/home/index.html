X-ray Lung Infection Detection

This utility will help doctors/Radiologist detection of Lung infection for Multiple frontal chest X-RAYS as we implement the trained model for More than one Image File.​This Lung infection detection works on sequential Deep learning model.


This module will serve in detection of lung infection and will be beneficial to X-RAY radiologist as their time on observing lung patients X-RAY will get saved.This Lung infection detection works on sequential Deep learning model, Trained on TensorFlow GPU with around 500,000 parameters for classification of 3 classes NORMAL | PNEUMONIA | COVID-19.

Realtime Social Distancing

Our manuscript focuses on detecting if the people around are maintaining social distancing or not. Using our own self developed for detecting the frame of a person and displaying labels, they are marked as safe or unsafe if the distance is less than a certain value. This system can be used for monitoring people via video surveillance in CCTV. ​

From this module Social Distancing violations will get detected and warning highlight will be shown, along with the person who is violating the social distancing Rule.​This module is also based on deep learning object detection. The only thing we used here are pretrained weights from yolo darknet as which is used for people detection.

Facemask Detection

This Module is able to detect if the person is wearing face mask or not, it will be good for the surveillance were wearing facemask is mandatory specially in this COVID pandemic situation wearing face mask is Necessary so the surveillance manager can warn those who aren’t wearing facemask without need of observing everyone face.​


In this module, we will be developing a face mask detector which can prove to be beneficial in distinguishing between faces with masks and faces with no masks. In this report, we have proposed a neural network to classify presence of a face mask.​This Face Mask detection module works on Deep learning Sequential model trained on TensorFlow GPU with 14,671,252 parameters for classification of two classes, Masked and unmasked.
